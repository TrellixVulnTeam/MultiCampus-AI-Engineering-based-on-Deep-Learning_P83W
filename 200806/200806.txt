# 웹프로그래밍(태그)
meta : 페이지에 대한 부가적인 설명. 실제 실행되지는 않음
link : 선택하면 다른 페이지로 이동
script : 팝업창
body : 실질적으로 웹사이트에 표시되는 코드가 시작
div : 영역을 나눔
table : 표를 나타내는 태그 (tr, th 등으로 행, 열 구분)
